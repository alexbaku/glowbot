# GlowBot.AI

An AI-powered skincare consultant that provides personalized routines and recommendations via WhatsApp.

## Features

- Skin analysis through photo processing
- Personalized skincare recommendations
- WhatsApp integration for easy access
- Ongoing progress tracking
- Customized routines based on user goals

## Tech Stack

- Python 3.9+
- FastAPI
- PostgreSQL
- Claude API
- Twilio WhatsApp API
